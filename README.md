# `@cinevva/usdjs`

Detachable, engine-agnostic **USD** (Universal Scene Description) runtime for the browser.

## Scope (current)

- Foundations only:
  - `SdfPath` implementation
  - minimal layer data structures
  - USDA lexer skeleton
  - Worker RPC shim

## Non-goals (for now)

- USDZ
- Full composition semantics (Pcp)
- Full schema implementations (UsdGeom/UsdShade/UsdPhysics)
- Three.js adapter

## Build

```bash
cd packages/usdjs
npm run build
```
