# Playground

Launch the full `@cinevva/usdjs-viewer` to see USD rendering in action.

<UsdPlayground />

## Features

**Full rendering**: Materials, textures, lights, and skeletal animation via Three.js.

**Multiple input modes**: Drag-and-drop files, paste USDA, or browse curated test assets.

**Scene inspection**: Prim hierarchy tree, property inspector, and animation controls.

**Format support**: USDA, USDC, and USDZ files all work.

## What You Can Do

- **Drop a file**: Drag any `.usd`, `.usda`, `.usdc`, or `.usdz` file onto the viewer
- **Paste USDA**: Click the editor button and paste USDA text directly
- **Browse corpus**: Explore test files from USD-WG, NVIDIA, and FT-Lab samples

For more details, see the [viewer documentation](https://cinevva-engine.github.io/usdjs-viewer/).
